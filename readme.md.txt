readme for  StudentFormViewer and Student ID Form Frame

compile both using "javac StudentIDFormViewer.java StudentFormFrame.java" (w/o quotes)

you can resize using corners of window

ENTERING STRING INPUTS INTO TEXTFIELD MUST BE FOLLOWED BY AN ENTER OR THE INPUT WONT BE REGISTERED

Sample Input:

"Robin Lopez " in text field followed by pressing enter
Press Year 1
Choose ITEC
Press CPS101
Press Done

Sample Output would be : Robin Lopez is a Year 1 ITEC student that is taking CPS 101
